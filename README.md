# Hibernate-Employee-mgt-system-project
This is a simple Hibernate-based Java application that demonstrates how to perform database operations using Hibernate ORM instead of traditional JDBC.
The project focuses on configuring Hibernate with MySQL and understanding how Java objects are mapped to database tables.
